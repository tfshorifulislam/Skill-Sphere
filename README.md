# Project Name: Skill Sphere

## 📌 Purpose
This project is a modern web application built with Next.js that provides an interactive platform for learning, browsing courses, and exploring instructors. It is designed to deliver a smooth user experience with fast performance, responsive UI, and dynamic data handling from APIs.

---

## 🚀 Live URL
https://skill-sphere-topaz.vercel.app/
---

## ✨ Key Features
- 📚 Dynamic course listing from API
- 🔍 Search and filtering functionality
- 👨‍🏫 Instructor showcase section
- ⚡ Fast performance with Next.js App Router
- 🎨 Modern UI using HeroUI components
- 🔔 Toast notifications for better UX
- 🖼️ Animated UI elements (Animate.css)
- 🔄 Smooth scrolling marquee effects
- 🔐 Authentication system (Better Auth + MongoDB adapter)
